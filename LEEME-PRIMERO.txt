📋 LO QUE INCLUYE:

  ✅ Frontend completo con calculadora de metros
  ✅ Sistema de checkout con upload de archivos
  ✅ Panel de admin funcional
  ✅ Gestión de pedidos y usuarios
  ✅ Descarga de archivos de diseño
  ✅ API REST completa
  ✅ Base de datos con Prisma
  ✅ Autenticación segura (NextAuth)
  ✅ Sistema de estados de pedidos
  ✅ Diseño responsive

🎯 FLUJO DE USO:

  Cliente:
  1. Selecciona metros en calculadora
  2. Sube archivo de diseño
  3. Completa formulario
  4. Realiza pedido

  Admin:
  1. Login en /auth/signin
  2. Ve pedidos en /admin/pedidos
  3. Descarga archivos de diseño
  4. Cambia estados
  5. Añade número de seguimiento

🔧 CONFIGURACIÓN:

  Edita el archivo .env con tus datos de PostgreSQL:
  DATABASE_URL="postgresql://user:password@localhost:5432/dtf_print_services"

💡 PRÓXIMOS PASOS:

  1. Configura .env con tu base de datos
  2. Ejecuta ./START.sh o los comandos manuales
  3. Abre http://localhost:3000
  4. ¡Empieza a vender film DTF!

🆘 SOPORTE:
